# Методы примитивов

Кажется логичным, что методы и свойства могут быть только у сложных структур вроде объектов, однако язык предоставляет возможность вызывать методы и у самых обычных чисел или строк. Понимание того, как работать с числами, правильно их округлять, находить минимальные и максимальные значения, а также извлекать чистые данные из строк с единицами измерения — это базовая и ежедневная рутина фронтенд-разработчика при создании логики интерфейсов.

### Как работают методы у примитивов

К примитивным типам в JavaScript относятся строки, числа, логические значения и некоторые другие. Удивительной механикой языка является то, что при попытке вызова метода у примитива под капотом создается специальный временный объект-обертка. Этот объект содержит в себе исходное значение и предоставляет доступ к набору встроенных вспомогательных функций. Сразу после того как вызванный метод отрабатывает и возвращает результат, этот временный объект удаляется из памяти. Весь процесс происходит молниеносно, позволяя нам манипулировать базовыми значениями так, словно они являются полноценными объектами.

### Основные методы чисел

При работе с дробными значениями (например, при расчете цен) часто возникает необходимость ограничить количество знаков после запятой. Для этого используется метод `toFixed`. Он округляет значение до нужного количества знаков и возвращает результат в формате строки. Если вызвать метод без передачи аргументов, число будет округлено до целого.

```javascript
const price = 99.99;
console.log(price.toFixed(1)); // Выведет строку "100.0"
console.log(price.toFixed());  // Выведет строку "100"
```

Если необходимо вызвать метод непосредственно на самом числе, а не через переменную, синтаксис требует внимания. Нужно использовать либо две точки подряд `5..toFixed(2)`, либо обернуть число в круглые скобки `(5).toFixed(2)`. Иначе интерпретатор воспримет единственную точку как начало дробной части числа и выдаст ошибку.

Метод `toPrecision` похож на предыдущий, но он форматирует число до строго определенной длины, начиная отсчет не после точки, а с самой первой цифры всего числа.

Для явного перевода числа в строковый тип применяется метод `toString`. Его интересная особенность — возможность передать аргумент, указывающий систему счисления. Например, передав двойку `num.toString(2)`, можно получить строковое представление числа в двоичном формате.

### Математические вычисления с объектом Math

Для выполнения сложных вычислений в языке предусмотрен глобальный встроенный объект `Math`. Он хранит в себе популярные математические константы и полезные функции, которые избавляют разработчика от необходимости писать собственные алгоритмы.

Для генерации случайного дробного числа от нуля до единицы применяется метод `Math.random()`. Если требуется получить модуль числа (его абсолютное значение без знака минус), используется `Math.abs()`. Для возведения числа в нужную степень существует функция `Math.pow()`, где первым аргументом передается базовое число, а вторым — степень.

Поиск минимального или максимального значения в наборе чисел осуществляется через функции `Math.min()` и `Math.max()`. Особенность этих функций в том, что они принимают аргументы строго через запятую. Если ваши числа хранятся в массиве, перед передачей массив необходимо «распаковать» при помощи spread-оператора (трех точек).

```javascript
const numbers = [10, 5, 100, 34];
console.log(Math.max(...numbers)); // Вернет 100
```

### Округление чисел

Объект `Math` предоставляет четыре метода для округления чисел, работающих по разным сценариям. Метод `Math.round()` округляет число до ближайшего целого по стандартным математическим правилам. Если требуется принудительно округлить значение вниз, используется `Math.floor()` (пол), а для округления всегда в большую сторону — `Math.ceil()` (потолок). Также есть метод `Math.trunc()`, который просто отбрасывает дробную часть числа без какого-либо округления и без учета знака.

### Извлечение чисел из строк

В веб-разработке данные часто приходят в виде строк, содержащих помимо чисел различные единицы измерения (например, `"100px"` или `"15.5rem"`). Базовые методы приведения типов, такие как функция `Number()` или унарный плюс `+`, не справятся с этой задачей и вернут `NaN` (Not a Number).

Для безопасного извлечения числовых данных из таких строк используются функции `parseInt` и `parseFloat`. Функция `parseInt` анализирует строку слева направо и достает из нее целое число, игнорируя последующие буквы. Если же вам необходимо сохранить дробную часть, применяется `parseFloat`. Важное условие: число должно находиться в самом начале строки. Если первым символом окажется буква, парсинг прервется и функция вернет `NaN`.

```javascript
const elementWidth = "150.5px";
console.log(parseInt(elementWidth));   // Вернет 150
console.log(parseFloat(elementWidth)); // Вернет 150.5
```

### Итог

Мы разобрали скрытый механизм работы методов у примитивов и изучили ключевые инструменты для взаимодействия с числами. Вы познакомились с форматированием чисел с помощью методов `toFixed` и `toString`, научились применять мощный встроенный объект `Math` для математических операций и округления, а также освоили безопасный парсинг чисел из сложных строк с помощью `parseInt` и `parseFloat`. Эти инструменты составляют надежную базу для любых вычислений в клиентском коде.

### Задача на закрепление

Напишите скрипт, который рассчитывает финальную стоимость товара с учетом скидки и подготавливает значение для вывода в корзине пользователя.

У вас есть переменная-строка `priceString` со значением `"540.75$"` и переменная `discount` со значением `15` (процент скидки). Необходимо извлечь числовое значение цены из строки, рассчитать итоговую стоимость с учетом скидки, округлить полученный результат до целого числа по математическим правилам и вывести строку формата: `Финальная цена: 460`.

**Критерии успешного выполнения:**

* Для получения начальной цены из строки использована правильная функция парсинга, которая сохраняет дробную часть и отбрасывает символ валюты.
* Вычисление скидки реализовано с помощью математических операторов.
* Округление итогового значения выполнено через соответствующий метод объекта `Math`.
* В консоль выводится корректная строка с правильным числом.
